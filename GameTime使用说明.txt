                    GameTime 使用说明
═══════════════════════════════════════════════════════════

【概述】

  GameTime 是一款家长控制应用，通过"多邻国学习 → 赚取游戏时间"的机制，
  让孩子用学习时间换取游戏时间。应用作为无障碍服务运行，无法被常规方式关闭，
  强制关闭将触发惩罚和家长邮件通知。


═══════════════════════════════════════════════════════════
一、核心机制
═══════════════════════════════════════════════════════════

  1.1 学习与游戏兑换

    - 学习 1 分钟 = 获得 1 分钟游戏时间（1:1 兑换）
    - 学习应用：多邻国 (com.duolingo)
    - 学习时自动检测，每 60 秒结算一次
    - 进入游戏时自动停止学习计时，反之亦然（互斥）

  1.2 连续打卡加成

    连续 1-6 天        0%
    连续 7-13 天      +10%
    连续 14-20 天     +15%
    连续 21-27 天     +20%
    连续 28 天以上    +25%

    - 周六/周日（节假日）额外 2 倍兑换加成
    - 每日打卡门槛：最少学习 10 分钟才算有效打卡
    - 图标：🥢 已打卡 / 💩 未打卡

  1.3 游戏时间限制

    工作日每日限额    30 分钟
    节假日每日限额    60 分钟
    可玩游戏时段      08:00 - 22:30

    - 超出限额或时段 → 全屏遮罩拦截
    - 余额不足 → 同样拦截

  1.4 默认封锁游戏列表

    王者荣耀          com.tencent.tmgp.sgame
    和平精英          com.tencent.tmgp.pubgmhd
    原神              com.miHoYo.GenshinImpact
    崩坏：星穹铁道    com.HoYoverse.hkrpgoversea
    明日方舟          com.hypergryph.arknights
    金铲铲之战        com.tencent.jkchess
    蛋仔派对          com.netease.party
    光遇              com.netease.sky / com.tgc.sky
    233乐园           com.meta.box

二、界面说明
═══════════════════════════════════════════════════════════

  2.1 主页面（仪表盘）

    - 今日学习 / 今日游戏时长
    - 连续打卡天数 + 加成百分比
    - 可用余额（游戏时间为 0 时不可玩）
    - 每日限额进度条
    - 里程碑：1 / 3 / 7 / 14 / 21 / 30 / 60 / 90 / 180 / 365 天
    - 日历图：苹果 Screen Time 风格，横向滚动
      绿色 = 学习，红色 = 游戏，Y轴上限 1 小时
    - 提示：学习未满 10 分钟时显示"今天学满10分钟打卡"

  2.2 设置页面（需 PIN 码）

    - 默认 PIN 码：2525
    - 工作日 / 节假日游戏时间限制
    - 邮箱报告配置（SMTP 发件人、授权码、收件人、发送时间）
    - 游戏封锁列表增删

  2.3 日志页面

    - 学习开始/结束、游戏启动/关闭/拦截
    - 显示应用中文名称和精确时间
    - 进程杀死记录 (APP_KILLED)


═══════════════════════════════════════════════════════════
三、邮箱日报
═══════════════════════════════════════════════════════════

  3.1 配置方法

    1. 设置页面 → 输入发件人 QQ 邮箱
    2. 输入 QQ 邮箱授权码
       （QQ邮箱 → 设置 → 账户 → POP3/SMTP服务 → 生成授权码）
    3. 输入家长收件人邮箱
    4. 设置发送时间（默认 22:00）
    5. 开启邮件报告开关

  3.2 报告内容（HTML 格式）

    - 学习 / 游戏时长对比卡片
    - 连续打卡天数 + 加成
    - 每小时分布柱状图（绿 = 学习，红 = 游戏）
    - 最近 10 条活动记录
    - 可用余额

四、安全机制
═══════════════════════════════════════════════════════════

  4.1 服务停止告警

    守护进程每 5 分钟检查无障碍服务状态。服务停止时：
    - 系统通知栏弹出"GameTime 服务异常"
    - 发送邮件告警给家长
    - 每次停止仅发送一次（重启后重置）

  4.2 进程杀死检测与惩罚

    服务每 30 秒写入心跳。检测到被手动杀死后：

    ┌────────────┬──────────────────────────────────┐
    │  第 1 次   │ 警告弹窗 + 邮件通知，不扣时间   │
    ├────────────┼──────────────────────────────────┤
    │  第 2 次起 │ 扣除游戏时间                     │
    │            │ 惩罚 = 杀死时长（分钟）           │
    │            │ 上限 = 当日限额                   │
    │            │ 可扣至负值，需学习还债             │
    └────────────┴──────────────────────────────────┘

    - 每周一自动重置计数
    - 手机关机 / 没电自动关机 → 不计入惩罚
      （通过 SystemClock.elapsedRealtime 区分重启与强制停止）


═══════════════════════════════════════════════════════════
五、安装与保活
═══════════════════════════════════════════════════════════

  5.1 首次安装

    1. 安装 APK
    2. 打开应用 → 点击"去开启"进入无障碍设置
    3. 找到 GameTime 服务 → 开启
    4. 授予"显示在其他应用上层"权限（悬浮拦截需要用）

  5.2 保活措施

    - 前台服务（通知栏持续显示）
    - 电池优化白名单
    - Doze 模式禁用
    - 5 分钟守护闹钟（AlarmManager）
    - 服务销毁时自动重新调度

  5.3 MIUI 额外设置

    设置 → 应用 → GameTime：
      - 自启动：允许
      - 电池：无限制
    多任务界面 → 长按 GameTime → 锁定

六、常见问题
═══════════════════════════════════════════════════════════

  Q：游戏计时不准确？
  A：已采用挂钟模式，按实际经过时间计算，不受系统延迟影响。

  Q：学习和游戏同时在后台怎么办？
  A：互斥设计，进入学习停止游戏计时，反之亦然。

  Q：孩子卸载应用怎么办？
  A：作为设备管理器运行后无法直接卸载，需先取消激活。

  Q：PIN 码忘记了怎么办？
  A：清除应用数据可重置为默认值 2525（也会清除历史记录）。
