# GameTime - 学习解锁游戏时间

通过多邻国学习赚取游戏时间，家长控制工具。

## 功能

- **学习追踪**：通过无障碍服务自动监测多邻国学习时长
- **游戏拦截**：基于包名匹配拦截游戏应用，余额不足时全屏遮挡
- **兑换公式**：1分钟学习 = 1分钟游戏 × (1 + 连续打卡加成) × 周末双倍
- **连续打卡**：每日 +5% 加成，最高 +25%
- **每日限额**：工作日/节假日独立可配置（默认 60/120 分钟）
- **时间窗口**：游戏仅在 8:00-22:30 可用
- **PIN 保护**：4 位数字密码锁保护设置访问
- **活动日志**：记录学习/游戏/拦截事件
- **邮件报告**：每日 QQ 邮箱 SMTP 定时发送
- **小时级日历**：24 小时柱状图显示学习和游戏分布
- **防卸载**：设备管理器激活后阻止卸载
- **服务守护**：每 15 分钟检测无障碍服务状态，异常时通知 + 邮件告警

## 技术栈

- Kotlin + Jetpack Compose + Material 3
- Android SDK 34+ (compileSdk 36)
- AccessibilityService 核心服务
- SharedPreferences 数据持久化
- JavaMail (QQ SMTP) 邮件发送
- AlarmManager 定时任务
- Canvas 自绘图表
